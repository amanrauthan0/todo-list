Todo List App (React)

A simple yet powerful Todo List application built while learning React.
This project goes beyond basic todos by allowing notes, checklists, and persistent storage.

🚀 Features

➕ Add and delete todos

✏️ Edit notes (“About Todo”) for each task

✅ Checklist support inside notes

📌 Mark tasks as completed with visual feedback

💾 Data persistence using localStorage

🎨 Clean and responsive UI with Tailwind CSS

🧠 What I Learned

This project helped me understand:

React state management using useState

Immutable updates for arrays and objects

Conditional rendering

Controlled inputs and textareas

Handling checkboxes properly

Saving and loading data from localStorage

Structuring components for scalability

🛠 Tech Stack

React

JavaScript (ES6+)

Tailwind CSS
